Create Natural Party¶

  1. Entering data manually using a form

Choose this option if you want to create a Natural Party using a form.

Natural Party acting as a party.

  1. From another QGIS layer/table (setting a field mapping)

Choose this option to open a window that allows you to import data from a source table into the LADM_COL interesado_natural table.

If the field structure of input and target tables differs, you can set a field mapping to define field transformations and correspondence.
