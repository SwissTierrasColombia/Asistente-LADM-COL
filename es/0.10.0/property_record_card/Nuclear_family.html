Núcleo Familiar¶

Crear Núcleo Familiar¶

  1. Ingresando datos manualmente en un formulario

Elige esta opción si deseas agregar un Núcleo Familiar con un formulario.

Núcleo Familiar permite registrar la información relativa a los núcleos familiares del levantamiento catastral de los pilotos del Catastro Multipropósito.

  1. Desde otra capa de QGIS (definiendo un mapeo de campos)

Elige esta opción para abrir una ventana que te permite importar datos desde una tabla fuente hacia la tabla nucleofamiliar de LADM_COL.

Si la estructura de campos de las tablas de entrada y salida difiere, puedes definir un mapeo para transformar campos y establecer correspondencias entre ellos.

Para usar esta función revisa este ENLACE.
