Equipamento comunal¶

Crear equipamento comunal¶

  1. Ingresando datos manualmente en un formulario

Elige esta opción si deseas agregar una Equipamento comunal con un formulario.

La clase**Equipamento comunal** es para registrar la información relativa al equipamiento comunal del predio matriz de la propiedad horizontal

  1. Desde otra capa de QGIS (definiendo un mapeo de campos)

Elige esta opción para abrir una ventana que te permite importar datos desde una tabla fuente hacia la tabla equipamiento_comunal de LADM_COL.

Si la estructura de campos de las tablas de entrada y salida difiere, puedes definir un mapeo para transformar campos y establecer correspondencias entre ellos.

Para usar esta función revisa este LINK.
