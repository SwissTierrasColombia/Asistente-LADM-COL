Geoeconomic Zone¶

Create Geoeconomic Zone¶

  1. Entering data manually using a form

Choose this option if you want to create a Geoeconomic Zone using a form.

Geoeconomic Zone this class allows you to manage areas with similar economic characteristics and geographical location, defined to characterize the valuation.

  1. From another QGIS layer/table (setting a field mapping)

Choose this option to open a window that allows you to import data from a source table into the LADM_COL zona_homogenea_geoeconomica table.

If the field structure of input and target tables differs, you can set a field mapping to define field transformations and correspondence.

For use this function check this LINK.
